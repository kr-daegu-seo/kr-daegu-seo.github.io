# 서구 · 대구광역시

> 서구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-daegu-seo.github.io/**

대구광역시 서구는 낙동강 지류인 금호강이 북서쪽을 감싸 흐르고 와룡산 자락이 남서 경계를 이루는 면적 17.48㎢의 도심형 자치구입니다. 조선시대 대구부 서중면과 달서면 일대가 뿌리이며, 비산동·평리동·원대동·내당동·상중이동을 아우르는 9개 법정동과 17개 행정동으로 짜여 있습니다. 무엇보다 이 지역은 대구를 '섬유도시'로 각인시킨 염색·직물 산업의 본산으로, 대구염색산업단지와 서대구산업단지가 지역 경제의 뼈대를 이루어 왔습니다. 2022년 서대구역이 KTX·SRT 정차역으로 개통하면서 서대구역세권 개발이 본격화되어 낡은 공단 이미지를 벗고 새로운 도시 축으로 탈바꿈하는 중입니다. 달성토성마을의 정겨운 골목 벽화와 와룡산 용미봉의 진달래 군락은 산업의 도시에 서정을 더하는 서구만의 얼굴입니다.
